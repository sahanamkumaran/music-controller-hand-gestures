# Music Controller using Hand Gestures

## Overview
A real-time computer vision application that controls music playback using hand gestures captured through a webcam.

## Technologies Used
- Python
- OpenCV
- MediaPipe
- PyAutoGUI

## Features
- Real-time hand tracking
- Gesture recognition
- Play/Pause control
- Volume Up/Down control
- Touchless media interaction

## Workflow
1. Capture webcam feed
2. Detect hand landmarks using MediaPipe
3. Recognize predefined gestures
4. Execute media commands using PyAutoGUI

## Applications
- Touchless media control
- Smart home interfaces
- Human-computer interaction systems
